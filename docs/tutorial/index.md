# 튜토리얼 — IFRS 17과 fastcashflow

이 튜토리얼은 **IFRS 17을 처음 접하는 보험 실무자**를 대상으로,
보험계약부채를 측정한다는 것이 무엇인지부터 시작해 fastcashflow
엔진이 그 측정을 실제로 어떻게 수행하는지까지를 설명합니다.

수식과 코드가 등장하지만, 핵심은 **개념과 계산의 흐름**입니다.
파이썬을 몰라도 따라올 수 있도록, 코드는 "엔진에 무엇을 시키는가"를
보여주는 최소한의 호출 예시로만 씁니다.
