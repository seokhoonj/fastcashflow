# 자산·지급여력 5 — 지급여력비율

:::{admonition} 이 챕터에서 배우는 것
:class: tip

- **지급여력비율** = 가용자본 나누기 요구자본 (`fcf.solvency_ratio` / `assess_solvency`)
- 분자 (가용자본, 3 챕터) 와 분모 (요구자본, 4 챕터) 가 만나는 자리
- Solvency II 와 K-ICS 비율의 해석, 그리고 공시 숫자와의 연결
:::

> **작성 예정 (skeleton)** — 윤곽만 잡혀 있습니다.

## 다룰 내용

- `assess_solvency` 의 비율 산출 (가용자본 / 총 SCR)
- 자산위험이 큰 책에서 부채측-only 비율이 왜 상한인지 (자산측 통합의 의미)
- 비율 민감도 -- 다음 챕터 (대량해지 경감) 가 이 비율을 어떻게 움직이는지의 예고

## 관련 API / 쿡북

- `fcf.assets.assess_solvency`, `fcf.solvency_ratio`
